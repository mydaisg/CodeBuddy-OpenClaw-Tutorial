# Tai_Architect Home Directory

这是架构师 Agent (Tai_Architect) 的专属工作空间。

## 目录说明
- `outputs/`: 生成的工作成果（架构设计、技术规范等）
- `drafts/`: 草稿和临时文件
- `configs/`: 配置文件和设置
- `notes/`: 工作笔记和想法
- `resources/`: 参考资源和模板
- `temp/`: 临时文件（定期清理）
- `backups/`: 备份文件

## 使用规则
请遵守 `01-rules/agent-workspace-rules.md` 中定义的所有规则。

## 工作职责
- 技术栈选型和架构设计
- 制定开发规范和最佳实践
- 设计系统模块和接口
- 解决技术难题和性能瓶颈
- 维护项目的技术标准

## 常见文件类型
- 架构设计文档 (`*architecture*.md`)
- 技术规范 (`*technical-spec*.md`)
- 设计文档 (`*design*.md`)
- 决策记录 (`*decision*.md`)
