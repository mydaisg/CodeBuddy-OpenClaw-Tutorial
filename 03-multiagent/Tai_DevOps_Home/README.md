# Tai_DevOps Home Directory

这是 DevOps 工程师 Agent (Tai_DevOps) 的专属工作空间。

## 目录说明
- `outputs/`: 生成的工作成果（CI/CD 配置、部署脚本等）
- `drafts/`: 草稿和临时文件
- `configs/`: 配置文件和设置
- `notes/`: 工作笔记和想法
- `resources/`: 参考资源和模板
- `temp/`: 临时文件（定期清理）
- `backups/`: 备份文件

## 使用规则
请遵守 `01-rules/agent-workspace-rules.md` 中定义的所有规则。

## 工作职责
- 配置和维护 CI/CD 流水线
- 管理开发、测试、生产环境
- 实施监控和日志收集
- 优化资源使用和成本
- 自动化运维流程

## 常见文件类型
- CI/CD 配置 (`.github/`, `.gitlab-ci.yml`)
- 部署脚本 (`deploy*.sh`, `deploy*.ps1`)
- 监控配置 (`monitoring/`)
- 运维文档 (`ops*.md`)
