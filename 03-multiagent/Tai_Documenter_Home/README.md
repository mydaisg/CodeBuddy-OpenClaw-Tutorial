# Tai_Documenter Home Directory

这是文档工程师 Agent (Tai_Documenter) 的专属工作空间。

## 目录说明
- `outputs/`: 生成的工作成果（API 文档、用户手册等）
- `drafts/`: 草稿和临时文件
- `configs/`: 配置文件和设置
- `notes/`: 工作笔记和想法
- `resources/`: 参考资源和模板
- `temp/`: 临时文件（定期清理）
- `backups/`: 备份文件

## 使用规则
请遵守 `01-rules/agent-workspace-rules.md` 中定义的所有规则。

## 工作职责
- 生成和维护 API 文档
- 编写用户手册和操作指南
- 维护项目知识库
- 沉淀最佳实践和经验
- 确保文档的及时性和准确性

## 常见文件类型
- API 文档 (`api-docs/`, `swagger/`)
- 用户手册 (`user-guide*.md`)
- 技术文档 (`technical-docs/`)
- 知识库文章 (`knowledge-base/`)
