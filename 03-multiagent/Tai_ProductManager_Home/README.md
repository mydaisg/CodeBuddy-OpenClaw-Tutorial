# Tai_ProductManager Home Directory

这是产品经理 Agent (Tai_ProductManager) 的专属工作空间。

## 目录说明
- `outputs/`: 生成的工作成果（需求文档、产品规划等）
- `drafts/`: 草稿和临时文件
- `configs/`: 配置文件和设置
- `notes/`: 工作笔记和想法
- `resources/`: 参考资源和模板
- `temp/`: 临时文件（定期清理）
- `backups/`: 备份文件

## 使用规则
请遵守 `01-rules/agent-workspace-rules.md` 中定义的所有规则。

## 工作职责
- 收集和分析用户需求
- 编写用户故事和需求文档
- 优先级管理和需求排序
- 与团队沟通产品愿景
- 制定产品路线图

## 常见文件类型
- 需求文档 (`*requirements*.md`)
- 用户故事 (`*user-story*.md`)
- 产品规划 (`*roadmap*.md`)
- 优先级列表 (`*priorities*.md`)
