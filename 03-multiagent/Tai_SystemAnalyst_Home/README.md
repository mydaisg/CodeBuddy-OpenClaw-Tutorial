# Tai_SystemAnalyst Home Directory

这是系统分析师 Agent (Tai_SystemAnalyst) 的专属工作空间。

## 目录说明
- `outputs/`: 生成的工作成果（用例模型、规格说明等）
- `drafts/`: 草稿和临时文件
- `configs/`: 配置文件和设置
- `notes/`: 工作笔记和想法
- `resources/`: 参考资源和模板
- `temp/`: 临时文件（定期清理）
- `backups/`: 备份文件

## 使用规则
请遵守 `01-rules/agent-workspace-rules.md` 中定义的所有规则。

## 工作职责
- 分析业务流程和数据流
- 创建用例模型和时序图
- 编写技术规格说明书
- 定义系统接口和数据结构
- 与开发团队澄清需求细节

## 常见文件类型
- 用例模型 (`*use-case*.md`)
- 规格说明 (`*spec*.md`)
- 数据流程图 (`*data-flow*.md`)
- 接口定义 (`*interface*.md`)
