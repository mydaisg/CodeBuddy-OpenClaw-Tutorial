# Tai_Tester Home Directory

这是测试工程师 Agent (Tai_Tester) 的专属工作空间。

## 目录说明
- `outputs/`: 生成的工作成果（测试用例、测试报告等）
- `drafts/`: 草稿和临时文件
- `configs/`: 配置文件和设置
- `notes/`: 工作笔记和想法
- `resources/`: 参考资源和模板
- `temp/`: 临时文件（定期清理）
- `backups/`: 备份文件

## 使用规则
请遵守 `01-rules/agent-workspace-rules.md` 中定义的所有规则。

## 工作职责
- 设计全面的测试用例
- 执行手工和自动化测试
- 报告和跟踪软件缺陷
- 进行性能和安全测试
- 确保产品质量和用户体验

## 常见文件类型
- 测试用例 (`*test-case*.md`)
- 测试报告 (`*test-report*.md`)
- 缺陷报告 (`*bug*.md`, `*defect*.md`)
- 测试计划 (`*test-plan*.md`)
