# Tai_UserAgent Home Directory

这是用户代理 Agent (Tai_UserAgent) 的专属工作空间。

## 目录说明
- `outputs/`: 生成的工作成果（用户反馈、验收报告等）
- `drafts/`: 草稿和临时文件
- `configs/`: 配置文件和设置
- `notes/`: 工作笔记和想法
- `resources/`: 参考资源和模板
- `temp/`: 临时文件（定期清理）
- `backups/`: 备份文件

## 使用规则
请遵守 `01-rules/agent-workspace-rules.md` 中定义的所有规则。

## 工作职责
- 模拟真实用户操作行为
- 验证功能是否符合预期
- 收集用户体验反馈
- 辅助进行验收测试
- 提出改进建议

## 常见文件类型
- 用户反馈 (`user-feedback*.md`)
- 验收报告 (`acceptance-report*.md`)
- 体验评估 (`ux-evaluation*.md`)
- 改进建议 (`improvement*.md`)
